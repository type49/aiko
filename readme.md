# [AiKo: The Privacy-First Local Assistant]

**AiKo** — это модульная экосистема персонального ассистента, работающая по принципу **Edge Computing**. Вся обработка голоса, логика и данные остаются на вашем оборудовании. Никаких облаков, никакой передачи данных третьим лицам, полная автономность. Спроектировано для работы в условиях ограниченных ресурсов и отсутствия сетевого соединения.

---

##  Ключевые преимущества

- **Offline-First:** Работает без интернета. Распознавание речи (Vosk) и синтез происходят локально.
    
- **Deep Telemetry:** Встроенный "черный ящик" и система самодиагностики для отслеживания ресурсов (CPU/RAM) и состояния потоков.
    
- **Hybrid Interface:** Управление голосом, через GUI или удаленно через Telegram.
    
- **QA-Ready:** Покрыта Unit-тестами и имеет строгую структуру логирования.
    

---

##  Архитектура

Проект построен на асинхронном ядре с разделением ответственности:

- **Core:** Координация сообщений, управление контекстом и состояниями.
    
- **Audio Engine:** Захват и обработка аудиопотока в реальном времени.
    
- **Plugin System:** Независимые модули, которые легко добавлять без изменения основного кода.
    
- **UI/UX:** Легкий Tray-интерфейс с системой всплывающих окон.
    

---

## Технологический стек

- **Язык:** Python 3.10+
    
- **GUI:** PySide6 (Qt for Python)
    
- **STT:** Vosk (Local Speech-to-Text)
